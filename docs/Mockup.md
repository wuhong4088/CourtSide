## Shared Design Style

- Clean and modern sports theme
- Same header and navigation on every page
- Blue-to-purple gradient for the hero section and primary accents
- White background with rounded cards
- Consistent buttons, form fields, spacing, and typography
- Primary color: `#2563EB`
- Secondary color: `#7C3AED`
- Accent color: `#22C55E`
- Background color: `#F8FAFC`
- Font: Inter
